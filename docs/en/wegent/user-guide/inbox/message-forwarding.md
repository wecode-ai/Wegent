# Message Forwarding

This document explains how to forward messages from conversations to other users or save them to your own queue.

---

## 📋 Forwarding Overview

The message forwarding feature allows you to share messages from conversations or save them for later. Three forwarding modes are supported:

| Mode | Description | Use Case |
|------|-------------|----------|
| **Forward to Others** | Send to other users' todo queues | Collaboration, task assignment, information sharing |
| **Save to Own Queue** | Save to your own todo queue | Process later, reminders, material collection |
| **Start New Chat** | Start a new conversation based on message content | Continue discussion, explore further |

---

## 🚀 Forwarding Entry Point

### Message Bubble Menu

On the chat page, each message bubble has a forward button:

1. Find the message you want to forward
2. Hover over the message bubble
3. Click the **Forward** button (or the forward option in the more menu)
4. The forward dialog opens

---

## 👥 Forward to Others

Send messages to other users' todo queues.

### Steps

1. Click the **Forward** button on the message
2. Select the **Forward to Others** tab in the forward dialog
3. Select recipients:
   - **Search Users**: Enter username or email in the search box
   - **Recent Contacts**: Select from the recent contacts list
4. Select target queue (optional):
   - By default, messages are sent to the recipient's default queue
   - You can select other public queues of the recipient
5. Add a note (optional)
6. Set priority:
   - **Low**: Non-urgent messages
   - **Normal**: Regular messages (default)
   - **High**: Urgent or important messages
7. Click **Send** to complete forwarding

### Recipient Selection

#### Search Users

- Search by username
- Search by email
- Search results display username and email

#### Recent Contacts

- Shows users you've recently forwarded messages to
- Sorted by most recent contact time
- Click to add as recipient

### Queue Selection

- Shows the recipient's public queue list
- Default queue is selected by default
- If the recipient has no public queues, messages will be sent to their default queue

---

## 📥 Save to Own Queue

Save messages to your own todo queue for later processing.

### Steps

1. Click the **Forward** button on the message
2. Select the **Save to Own Queue** tab in the forward dialog
3. Select target queue:
   - Your default queue is selected by default
   - You can select other queues
4. Add a note (optional)
5. Set priority
6. Click **Send** to complete saving

### Use Cases

- **Process Later**: No time to handle now, save to queue for later
- **Task Reminders**: Save important messages as to-do items
- **Material Collection**: Collect valuable conversation content
- **Categorization**: Save to different queues by topic

---

## 💬 Start New Chat

Start a new conversation based on the current message content.

### Steps

1. Click the **Forward** button on the message
2. Select the **Start New Chat** tab in the forward dialog
3. Click the **Start Chat** button
4. The system will navigate to the chat page with message content as context

### Use Cases

- **Continue Discussion**: Continue in-depth discussion on the current topic
- **Switch Agent**: Use a different agent to handle the same problem
- **New Perspective**: Explore the problem from a new angle

---

## 📝 Message Selection

You can select specific messages to forward.

### Select Single Message

- Click the forward button on the message directly
- The current message is selected by default

### Select Multiple Messages

In the forward dialog:

1. View the message list
2. Check the messages you want to forward
3. Use **Select All** or **Deselect All** for quick operations
4. The number of selected messages is displayed in the list header

### Message Preview

- Each message shows sender and time
- Message content preview is displayed (truncated)
- User messages and AI messages have different icon indicators

---

## ⚙️ Forwarding Options

### Note

- You can add a note to forwarded messages
- Notes are displayed in the recipient's message details
- Used to explain the reason for forwarding or provide additional information

### Priority Settings

| Priority | Description | Display Style |
|----------|-------------|---------------|
| **High** | Urgent or important messages | Red label |
| **Normal** | Regular messages | No special marking |
| **Low** | Non-urgent messages | Blue label |

---

## 💡 Best Practices

### Forwarding Etiquette

- Add clear notes explaining the reason for forwarding
- Set priority appropriately, avoid overusing high priority
- Choose appropriate recipients, avoid disturbing unrelated people

### Efficient Forwarding

- Use recent contacts to quickly select recipients
- Create dedicated queues for different types of messages
- Regularly organize and process received forwarded messages

### Team Collaboration

- Establish team forwarding guidelines
- Use consistent priority standards
- Promptly process received high-priority messages
