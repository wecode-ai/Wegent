# Project Work Chat

Project work chat starts a task in the context of a specific project. After a project is selected, Wework associates the new conversation with that project and shows the project name in the empty-state title so the active work context is clear.

## Enter Project Work

1. Open the Wework workbench.
2. Click **Enter project work** below the composer.
3. Select the target project from the project list.
4. The title changes to **What should we build in {project name}?**.

If the project list is empty, create a project in project management first.

## Send Project Messages

In project work chat, you can type a requirement or upload attachments and send without text. When sending attachments only, the system uses the default message **请参考附件** and submits the attachment IDs to the backend.

## Attachments

Click **+** to open the menu. The menu only contains **Add photos and files**. Uploaded images show thumbnail previews above the composer; other files show their file name and size. Attachments can be removed before sending.

## Models and Skills

Before creating a project conversation, you can choose a model and Skills:

- The model list only shows models whose name contains `claude`.
- Multiple Skills can be selected.
- After the conversation is created, the model and Skills are locked and cannot be changed again in the same task.

Follow-up messages continue to use the model and Skills selected when the task was created.
