## ðŸŽ¯ Core Features

### Task Management

Task is the executable work unit in Wegent that assigns user requirements to Teams or Bots for execution. You can:

- Create and configure tasks
- Monitor task execution status
- View task execution results
- Manage task lifecycle

### IM Channel Integration

Connect Wegent agents to instant messaging platforms, allowing users to interact with AI agents directly in familiar chat tools:

- DingTalk integration support
- Real-time streaming responses
- Multi-turn conversation support
- AI card display

### Requirement Clarification

Help users transform vague requirements into clear development tasks through interactive Q&A:

- Structured question guidance
- Smart option recommendations
- Final prompt generation
- One-click task creation
