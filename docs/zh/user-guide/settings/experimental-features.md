# Wework 实验性功能

Wework 可以隐藏仍在实验阶段、可能变化或不稳定的功能。实验性功能默认关闭，只有在当前设备上主动开启后才会显示。

## 开启实验性功能

1. 打开 Wework 桌面端的“设置”。
2. 进入“通用”。
3. 打开“实验性功能”开关。

设置会保存在当前设备上，并立即更新正在显示的界面。关闭开关后，实验性功能入口会再次隐藏。

目前 TODO 和站点属于实验性功能。默认情况下，Wework 左上角不显示 TODO 切换按钮，侧边栏也不显示“站点”入口；开启实验性功能后，TODO 按钮会出现在 Wework 与应用切换按钮之间，“站点”入口也会出现在侧边栏中。
