# 概述

AI Knowledge（AI 知识库）是 Wegent 的知识管理功能，让您可以创建和管理结构化知识，供 AI 智能体在对话中检索和引用。

---

## 📋 核心概念

### 知识库

**知识库**是存储和管理知识的容器：

- **文档集合**：包含多个相关文档
- **向量化存储**：文档内容转换为向量便于语义检索
- **权限控制**：支持私有和公开访问

### 检索器

**检索器**定义了如何从知识库中查找信息：

- **语义检索**：基于向量相似度的智能检索
- **关键词检索**：传统的关键词匹配
- **混合检索**：结合语义和关键词的优势

```
知识库 = 文档集合 + 向量化处理 + 元数据
检索器 = 检索策略 + 相关性算法 + 结果排序
```

---

## 🎯 主要功能

### 1. 知识库管理

创建和管理知识库：

- 创建多个知识库
- 上传和管理文档
- 自动文档分块和处理
- 查看处理状态和统计

### 2. 文档支持

支持多种文档格式和数据来源：

| 格式 | 说明 |
|------|------|
| **Markdown** | 推荐使用，保留格式和结构 |
| **PDF** | 自动提取文本内容 |
| **Word** | 支持 .docx 格式 |
| **纯文本** | .txt 文件 |
| **网页抓取** | 从 URL 自动抓取网页内容 |

#### 网页抓取

支持从网页直接抓取内容：

- 输入网页 URL 自动抓取
- 智能提取正文内容
- 支持动态网页渲染
- 自动处理分页内容

### 3. 检索配置

灵活配置检索行为：

- 选择检索策略（语义/关键词/混合）
- 设置返回结果数量
- 配置相关性阈值
- 自定义分块大小

### 4. 知识库应用

将知识库与智能体关联：

- 在智能体配置中选择知识库
- 对话时自动检索相关知识
- 查看引用来源

## 🚀 快速开始

### 创建第一个知识库

1. 进入 **Knowledge** 页面
2. 点击 **新建知识库**
3. 填写知识库信息：
   - 名称：如"产品文档"
   - 描述：知识库的用途说明
4. 上传文档文件
5. 等待文档处理完成

### 配置检索器

1. 进入知识库详情
2. 点击 **检索设置**
3. 选择检索策略：
   - 语义检索：适合概念性问题
   - 关键词检索：适合精确匹配
   - 混合检索：平衡两种方法
4. 调整参数并保存

### 在智能体中使用

1. 进入智能体设置
2. 在知识库选项中选择已创建的知识库
3. 保存配置
4. 与该智能体对话时将自动引用知识库内容
