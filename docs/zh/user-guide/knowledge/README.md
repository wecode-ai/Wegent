# 概述

AI Knowledge（AI 知识库）是 Wegent 的知识管理功能，让您可以创建和管理结构化知识，供 AI 智能体在对话中检索和引用。

## 📋 核心概念

### 知识库

**知识库**是存储和管理知识的容器：

- **文档集合**：包含多个相关文档
- **向量化存储**：文档内容转换为向量便于语义检索
- **权限控制**：支持私有和公开访问

### 知识库类型

Wegent 提供两种知识库类型：

| 类型 | 文档限制 | 对话功能 | 适用场景 |
|------|----------|----------|----------|
| **笔记本模式** | 最多 50 个文档 | ✅ 支持 | 小型知识库、交互式问答 |
| **经典模式** | 无限制 | ❌ 不支持 | 大型文档库、批量检索 |

### 检索器

**检索器**定义了如何从知识库中查找信息：

- **语义检索**：基于向量相似度的智能检索
- **关键词检索**：传统的关键词匹配
- **混合检索**：结合语义和关键词的优势

## 🎯 主要功能

### 1. 知识库管理

- 创建多个知识库
- 选择知识库类型（笔记本/经典）
- 上传和管理文档
- 自动文档分块和处理
- 查看处理状态和统计

### 2. 文档支持

支持多种文档格式和数据来源：

| 来源 | 说明 |
|------|------|
| **文件上传** | 支持 Markdown、PDF、Word、纯文本 |
| **文本粘贴** | 直接粘贴文本内容 |
| **外部表格** | 从钉钉/飞书导入表格 |
| **网页抓取** | 从 URL 自动抓取网页内容 |

### 3. 分块策略

灵活的文档分块选项：

| 策略 | 适用场景 |
|------|----------|
| **智能分块** | 通用文档，自动识别结构 |
| **句子分块** | FAQ、问答对，精确匹配 |
| **语义分块** | 长篇文章，保持主题连贯 |

### 4. 检索配置

- 选择检索策略（语义/关键词/混合）
- 设置返回结果数量
- 配置相关性阈值
- 检索测试功能

### 5. 摘要功能

- 文档级摘要：自动生成文档摘要
- 知识库级摘要：整体知识库概述
- 摘要重试：生成失败时可重试

## 🚀 快速开始

### 创建第一个知识库

1. 进入 **Knowledge** 页面
2. 点击 **新建知识库**
3. 选择知识库类型：
   - **笔记本模式**：适合小型知识库，支持对话
   - **经典模式**：适合大型文档库
4. 填写名称和描述
5. 上传文档文件
6. 等待文档处理完成

### 配置检索器

1. 进入知识库详情
2. 点击 **检索设置**
3. 选择检索策略：
   - 语义检索：适合概念性问题
   - 关键词检索：适合精确匹配
   - 混合检索：平衡两种方法
4. 使用 **检索测试** 验证效果
5. 调整参数并保存

### 在智能体中使用

1. 进入智能体设置
2. 在知识库选项中选择已创建的知识库
3. 保存配置
4. 与该智能体对话时将自动引用知识库内容

## 💡 使用场景

### 产品文档

- **用户手册**：存储产品使用说明
- **FAQ 集合**：常见问题及解答（推荐笔记本模式）
- **版本说明**：产品更新日志

### 技术文档

- **API 文档**：接口定义和使用示例
- **架构文档**：系统设计说明
- **最佳实践**：团队经验总结

### 企业知识

- **规章制度**：公司内部规范
- **培训材料**：员工培训文档（推荐笔记本模式）
- **项目文档**：历史项目资料（推荐经典模式）
