# 知识库类型

Wegent 支持两种知识库类型，满足不同使用场景的需求。

---

## 📚 类型对比

| 特性 | 笔记本模式 | 经典模式 |
|------|-----------|---------|
| **布局** | 三栏布局（对话+文档） | 仅文档列表 |
| **文档限制** | 最多 50 个 | 无限制 |
| **对话功能** | ✅ 支持 | ❌ 不支持 |
| **文档选择** | ✅ 支持 | ❌ 不支持 |
| **适用场景** | 交互式研究、问答 | 大型文档集合、档案 |

---

## 📓 笔记本模式

笔记本模式专为交互式场景设计，结合了文档管理和 AI 对话功能。

### 特点

- **三栏布局**：左侧文档列表、中间对话区、右侧详情面板
- **文档选择**：可选择特定文档作为对话上下文
- **AI 对话**：支持与智能体进行基于知识库的对话
- **实时引用**：AI 回答自动引用相关文档

### 适用场景

- 日常问答和研究
- 需要与 AI 交互的场景
- 团队协作讨论
- 基于文档的智能分析

### 文档限制

笔记本模式限制最多 **50 个文档**，以确保最佳性能和对话体验。

---

## 📁 经典模式

经典模式专注于文档管理，适合大型文档集合。

### 特点

- **简洁布局**：专注于文档浏览和管理
- **无数量限制**：可容纳大量文档
- **检索测试**：支持检索效果测试
- **文档管理**：完整的文档上传、编辑、删除功能

### 适用场景

- 大型文档存档
- 企业知识库
- 法规文档库
- 仅需要文档管理的场景

---

## 🔄 类型转换

您可以在两种类型之间进行转换。

### 笔记本 → 经典

- **条件**：无限制，随时可转换
- **影响**：
  - 保留所有文档
  - 对话历史仍然可见
  - 无法创建新的对话任务

### 经典 → 笔记本

- **条件**：文档数量 ≤ 50
- **影响**：
  - 保留所有文档
  - 启用对话功能
  - 可以创建对话任务

### 如何转换

1. 打开知识库详情页面
2. 点击 **转换类型** 按钮
3. 确认转换操作
4. 页面将自动刷新为新的布局

---

## 💡 选择建议

| 使用场景 | 推荐类型 | 原因 |
|----------|----------|------|
| 日常问答 | 笔记本 | 支持对话交互 |
| 文档存档 | 经典 | 无文档数量限制 |
| 团队协作 | 笔记本 | 结合对话和文档 |
| 大型知识库 | 经典 | 支持更多文档 |
| 快速检索 | 任意 | 两者都支持 RAG |
| 嵌入式使用 | 经典 | 适合作为外部知识源 |
