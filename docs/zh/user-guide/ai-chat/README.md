# 💬 AI 对话

AI 对话模块提供与 AI 智能体进行交互对话的功能，包括任务管理、IM 通道集成和需求澄清等。

---

|------|------|
| [管理任务](./managing-tasks.md) | 创建、管理和监控 AI 任务 |
| [IM 通道集成](./im-channel-integration.md) | 将 AI 智能体连接到钉钉等即时通讯平台 |

## 🎯 核心功能

### 任务管理

Task 是 Wegent 中的可执行工作单元，将用户需求分配给 Team 或 Bot 执行。您可以：

- 创建和配置任务
- 监控任务执行状态
- 查看任务执行结果
- 管理任务生命周期

### IM 通道集成

将 Wegent 智能体连接到即时通讯平台，让用户可以在熟悉的聊天工具中直接与 AI 智能体对话：

- 支持钉钉集成
- 实时流式响应
- 多轮对话支持
- AI 卡片展示

### 需求规范澄清

通过交互式问答帮助用户将模糊需求转化为清晰的开发任务：

- 结构化问题引导
- 智能选项推荐
- 最终 Prompt 生成
- 一键创建任务
