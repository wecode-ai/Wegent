# 项目工作对话

项目工作对话用于围绕一个具体项目发起任务。选中项目后，Wework 会把新对话关联到该项目，并在空态标题中显示项目名称，帮助确认当前工作的上下文。

## 进入项目工作

1. 打开 Wework 工作台。
2. 点击输入框下方的 **进入项目工作**。
3. 在项目列表中选择目标项目。
4. 标题会切换为 **我们应该在 {项目名称} 中构建什么？**。

如果项目列表为空，需要先在项目管理中创建项目。

## 发送项目消息

在项目工作对话中可以直接输入需求，也可以只上传附件后发送。仅发送附件时，系统会使用默认消息 **请参考附件**，并把附件 ID 一起提交给后端。

## 附件

点击 **+** 后仅提供 **添加照片和文件** 入口。上传图片后，输入框上方会显示缩略图预览；其他文件会显示文件名和大小。发送前可以移除已上传附件。

## 模型和技能

新建项目对话前可以选择模型和 Skill：

- 模型列表只展示名称中包含 `claude` 的模型。
- Skill 可以多选。
- 对话创建后，模型和 Skill 会锁定，不能在同一个任务中再次修改。

后续消息会继续使用任务创建时确定的模型和 Skill。
