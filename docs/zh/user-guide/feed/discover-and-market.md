# 发现与市场

Wegent 提供订阅器的发现和市场功能，让用户可以关注他人的公开订阅器或租用市场订阅器。

---

## 🔍 发现公开订阅器

### 进入发现页面

1. 进入 **Feed** 页面
2. 点击 **发现** 标签页

### 浏览订阅器

发现页面以卡片网格形式展示公开订阅器：

#### 卡片信息

| 元素 | 说明 |
|------|------|
| **名称** | 订阅器显示名称 |
| **类型标签** | 信息采集类或执行类 |
| **描述** | 订阅器的简要描述 |
| **最新执行** | 最近一次执行的状态和摘要 |
| **关注数** | 关注此订阅器的用户数 |
| **创建者** | 订阅器所有者的用户名 |

#### 排序方式

| 排序 | 说明 |
|------|------|
| **最热门** | 按关注数排序 |
| **最新** | 按创建时间排序 |

#### 搜索功能

在搜索框输入关键词搜索订阅器：

- 支持按名称搜索
- 支持按描述搜索
- 按 Enter 键执行搜索

### 查看执行历史

点击订阅器卡片打开历史记录弹窗：

- 查看最近的执行记录
- 查看 AI 生成的摘要
- 了解订阅器的实际效果

## ➕ 关注订阅器

### 关注操作

1. 在发现页面找到感兴趣的订阅器
2. 点击 **关注** 按钮
3. 按钮变为 **已关注** 状态

### 关注后的效果

- 该订阅器的执行动态会出现在您的时间线中
- 可以在"我关注的"列表中管理关注

### 取消关注

1. 点击 **已关注** 按钮
2. 按钮会显示取消关注的提示
3. 点击确认取消关注

---

## 📋 管理关注

### 查看关注列表

1. 进入订阅器管理页面
2. 切换到 **我关注的** 标签页

### 关注列表信息

| 信息 | 说明 |
|------|------|
| **订阅器名称** | 关注的订阅器名称 |
| **创建者** | 订阅器所有者 |
| **关注时间** | 开始关注的时间 |
| **任务类型** | 信息采集类或执行类 |

### 管理操作

- **查看详情**：跳转到订阅器详情页
- **取消关注**：停止关注该订阅器

---

## 🏪 订阅器市场

### 什么是市场订阅器

市场订阅器是所有者发布到市场供他人租用的订阅器：

- **租用者**可以使用订阅器的功能
- **租用者**可以自定义触发配置
- **租用者**可以使用自己的模型
- **Prompt 和智能体**由发布者设置，租用者不可见

### 进入市场

1. 进入 **Feed** 页面
2. 点击 **市场** 标签页

### 浏览市场订阅器

市场页面显示可租用的订阅器列表：

#### 列表信息

| 元素 | 说明 |
|------|------|
| **名称** | 订阅器显示名称 |
| **类型标签** | 信息采集类或执行类 |
| **描述** | 订阅器的功能描述 |
| **创建者** | 发布者用户名 |
| **租用次数** | 被租用的总次数 |
| **触发说明** | 原始触发配置描述 |

#### 排序方式

| 排序 | 说明 |
|------|------|
| **最多租用** | 按租用次数排序 |
| **最新** | 按发布时间排序 |

## 🔑 租用订阅器

### 租用流程

1. 在市场找到想要租用的订阅器
2. 点击 **租用** 按钮
3. 在弹出的配置对话框中设置参数
4. 点击 **租用** 确认

### 租用配置

#### 基本信息

| 字段 | 说明 |
|------|------|
| **显示名称** | 租用后的订阅器名称 |

#### 触发配置

可以自定义触发方式：

| 触发方式 | 说明 |
|----------|------|
| **定时计划** | 使用 Cron 表达式 |
| **固定间隔** | 按固定时间间隔 |
| **一次性定时** | 指定时间执行一次 |

> 📖 详细说明请参阅 [触发方式](./trigger-types.md)

#### 模型选择（可选）

可以选择使用自己的模型：

1. 点击模型选择器
2. 选择要使用的模型
3. 或保持默认使用发布者的模型

### 租用说明

租用订阅器时需要了解：

- **Prompt 由发布者设置**：您无法查看或修改 Prompt
- **使用发布者的智能体**：执行时使用发布者配置的智能体
- **可自定义触发和模型**：您可以控制何时执行和使用什么模型

---

## 📊 管理租用

### 查看租用列表

1. 进入订阅器管理页面
2. 切换到 **我租用的** 标签页

### 租用列表信息

| 信息 | 说明 |
|------|------|
| **租用名称** | 您设置的显示名称 |
| **来源订阅器** | 原始订阅器名称 |
| **发布者** | 原始订阅器所有者 |
| **触发配置** | 您设置的触发方式 |
| **状态** | 启用/禁用状态 |

### 管理操作

- **编辑**：修改触发配置和模型
- **启用/禁用**：控制是否执行
- **取消租用**：停止租用并删除

### 取消租用

1. 点击租用订阅器的 **取消租用** 按钮
2. 在确认对话框中点击 **确认**
3. 租用关系解除，执行记录保留

## 📤 发布到市场

### 发布条件

要将订阅器发布到市场：

1. 订阅器必须是您创建的
2. 订阅器配置完整且可正常执行

### 发布步骤

1. 编辑订阅器
2. 在可见性设置中选择 **市场**
3. 保存订阅器

### 发布后的效果

- 订阅器出现在市场列表中
- 其他用户可以租用
- 您可以看到租用统计

### 取消发布

1. 编辑订阅器
2. 将可见性改为 **私有** 或 **公开**
3. 保存订阅器

> ⚠️ 取消发布不会影响已有的租用关系

## 🤝 分享订阅器

### 邀请关注

订阅器所有者可以邀请特定用户关注：

1. 进入订阅器详情页
2. 点击 **分享** 按钮
3. 输入用户 ID 或邮箱
4. 发送邀请

### 处理邀请

收到邀请的用户：

1. 在通知中查看邀请
2. 点击 **接受** 或 **拒绝**
3. 接受后自动关注该订阅器

## 💡 使用建议

### 发现优质订阅器

- 关注热门订阅器了解最佳实践
- 查看执行历史评估订阅器质量
- 阅读描述了解订阅器用途

### 选择租用还是关注

| 场景 | 建议 |
|------|------|
| 想查看他人的执行结果 | 关注 |
| 想用自己的配置执行 | 租用 |
| 想学习订阅器设计 | 关注 |
| 想使用现成的自动化方案 | 租用 |

### 发布订阅器

- 编写清晰的描述说明用途
- 确保 Prompt 通用性好
- 定期检查执行效果

## ⚠️ 常见问题

### Q1：关注和租用有什么区别？

| 特性 | 关注 | 租用 |
|------|------|------|
| 查看执行结果 | ✅ | ✅ |
| 自定义触发配置 | ❌ | ✅ |
| 使用自己的模型 | ❌ | ✅ |
| 独立的执行记录 | ❌ | ✅ |

### Q2：租用的订阅器来源不可用怎么办？

如果原始订阅器被删除或取消发布：

- 已租用的订阅器仍可继续使用
- 但可能无法获取更新
- 建议联系发布者或寻找替代方案

### Q3：如何知道订阅器是否值得关注/租用？

评估方法：

1. 查看关注/租用数量
2. 查看最近的执行历史
3. 阅读描述了解用途
4. 查看创建者的其他订阅器
