# 💻 AI 设备

AI 设备模块允许您使用个人电脑（Mac、Linux 或 Windows）作为任务执行器，让 AI 任务直接在本地机器上运行。

## 🎯 核心功能

### 本地执行

将您的个人电脑注册为 Wegent 的任务执行器：

| 优势 | 描述 |
|------|------|
| **更低延迟** | 本地直接执行，无需网络传输延迟 |
| **数据隐私** | 您的代码和数据永远不会离开本地机器 |
| **环境控制** | 使用本地安装的工具、依赖和配置 |
| **成本节约** | 减少云端执行资源消耗 |
| **自定义设置** | 访问本地凭证、自定义工具和专业软件 |

### 设备管理

- **设备注册**：自动生成稳定的设备 ID
- **状态监控**：实时查看设备在线状态
- **并发控制**：每个设备支持最多 5 个并发任务
- **心跳机制**：自动维持连接和状态同步

### 安全机制

- **JWT 认证**：WebSocket 连接需要 token
- **用户隔离**：设备只能执行其所有者的任务
- **硬件绑定**：设备 ID 基于硬件标识

---

## 🚀 快速开始

1. **安装 Executor**：
   ```bash
   cd executor
   pip install -e .
   ```

2. **启动 Executor**：
   ```bash
   wegent-executor --mode local --token YOUR_JWT_TOKEN
   ```

3. **选择设备**：在聊天界面的设备选择器中选择您的本地设备

4. **执行任务**：发送消息，任务将在本地执行

---

## 📊 设备状态

| 状态 | 图标 | 描述 |
|------|------|------|
| **在线** | 🟢 | 设备已连接，有可用槽位 |
| **离线** | 🔴 | 设备未连接 |
| **繁忙** | 🟡 | 所有 5 个并发槽位均被占用 |
| **默认** | ⭐ | 您的新任务默认设备 |
