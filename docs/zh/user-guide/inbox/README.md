# 概述

待办是 Wegent 的消息转发与任务管理功能，让您可以将对话中的消息转发给其他用户或保存到自己的队列中，方便后续处理。

---

## 📋 核心概念

### 待办队列

**待办队列**是接收和管理转发消息的容器：

- **消息收集**：接收来自其他用户或自己转发的消息
- **状态管理**：跟踪消息的处理状态（未读、已读、处理中、已处理、已归档）
- **优先级标记**：支持高、普通、低三种优先级
- **可见性控制**：支持私有和公开两种可见性设置

### 消息转发

**消息转发**允许您将对话中的消息分享给他人或保存备用：

- **转发给他人**：将消息发送到其他用户的待办队列
- **保存到自己队列**：将消息保存到自己的待办队列中
- **开始新对话**：基于消息内容开始新的对话
---

## 🎯 主要功能

### 1. 队列管理

管理您的待办队列：

- 创建多个待办队列
- 设置队列名称和描述
- 配置队列可见性（私有/公开）
- 设置默认队列
- 删除不需要的队列

### 2. 消息转发

灵活的消息转发选项：

| 转发模式 | 说明 | 适用场景 |
|----------|------|----------|
| **转发给他人** | 发送到其他用户的队列 | 协作、任务分配 |
| **保存到自己队列** | 保存到自己的队列 | 稍后处理、备忘 |
| **开始新对话** | 基于消息开始新对话 | 继续讨论、深入探索 |

### 3. 消息管理

全面的消息管理功能：

- 查看消息详情和原始对话上下文
- 更新消息状态（已读/未读/已归档）
- 设置消息优先级
- 批量操作（批量标记、批量删除）
- 筛选和排序消息

### 4. 消息处理

将队列消息转化为新任务：

- 点击"处理"按钮开始处理消息
- 自动跳转到对话页面
- 消息内容作为上下文传递

## 🚀 快速开始

### 创建第一个待办队列

1. 点击左侧导航栏的 **待办** 入口
2. 点击右上角的 **+** 按钮
3. 填写队列名称和显示名称
4. 选择可见性设置：
   - **私有**：仅自己可见
   - **公开**：其他用户可以向此队列发送消息
5. 点击 **保存** 完成创建

### 转发第一条消息

1. 在对话页面，找到要转发的消息
2. 点击消息气泡上的 **转发** 按钮
3. 选择转发模式：
   - **转发给他人**：搜索并选择接收者
   - **保存到自己队列**：选择目标队列
   - **开始新对话**：直接开始新对话
4. 添加备注说明（可选）
5. 设置优先级
6. 点击 **发送** 完成转发

### 处理队列消息

1. 进入 **待办** 页面
2. 在左侧选择要查看的队列
3. 点击消息卡片查看详情
4. 点击 **处理** 按钮开始处理
5. 系统将跳转到对话页面，消息内容作为上下文

## 💡 使用场景

### 团队协作

- **任务分配**：将需要处理的消息转发给团队成员
- **信息共享**：将重要信息转发给相关人员
- **协作处理**：多人协作处理复杂任务

### 个人效率

- **稍后处理**：将暂时无法处理的消息保存到队列
- **任务备忘**：将重要消息保存为待办事项
- **信息整理**：按主题分类保存消息到不同队列

### 知识管理

- **收集素材**：将有价值的对话内容保存备用
- **案例积累**：保存典型问题和解决方案
- **经验分享**：将优质内容转发给团队成员
