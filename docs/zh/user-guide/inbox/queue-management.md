# 队列管理

本文档介绍如何创建和管理待办队列。

---

## 📋 队列概述

待办队列是接收和管理转发消息的容器。每个用户可以创建多个队列，用于分类管理不同类型的消息。

### 队列属性

| 属性 | 说明 |
|------|------|
| **名称** | 队列的唯一标识符，创建后不可修改 |
| **显示名称** | 队列的显示名称，可随时修改 |
| **描述** | 队列的用途说明 |
| **可见性** | 私有或公开 |
| **默认队列** | 是否为默认接收队列 |

### 可见性设置

| 可见性 | 说明 |
|--------|------|
| **私有** | 仅自己可以向此队列发送消息 |
| **公开** | 其他用户可以搜索并向此队列发送消息 |

---

## 🎯 创建队列

### 操作步骤

1. 进入 **待办** 页面
2. 点击侧边栏顶部的 **+** 按钮
3. 填写队列信息：
   - **名称**：输入队列的唯一标识符（仅支持字母、数字、下划线）
   - **显示名称**：输入队列的显示名称
   - **描述**：输入队列的用途说明（可选）
4. 选择可见性：
   - **私有**：仅自己可见
   - **公开**：其他用户可以向此队列发送消息
5. 点击 **保存** 完成创建

### 注意事项

- 队列名称创建后不可修改
- 建议使用有意义的名称，便于识别和管理
- 首次创建的队列会自动设为默认队列

---

## ✏️ 编辑队列

### 操作步骤

1. 在队列列表中找到要编辑的队列
2. 将鼠标悬停在队列上，点击右侧的 **更多** 按钮（三个点图标）
3. 选择 **编辑**
4. 修改队列信息：
   - 显示名称
   - 描述
   - 可见性
5. 点击 **保存** 完成修改

### 可修改的属性

- ✅ 显示名称
- ✅ 描述
- ✅ 可见性
- ❌ 名称（不可修改）

---

## ⭐ 设置默认队列

默认队列是接收转发消息时的首选队列。当其他用户向您转发消息时，如果未指定队列，消息将发送到您的默认队列。

### 操作步骤

1. 在队列列表中找到要设为默认的队列
2. 将鼠标悬停在队列上，点击右侧的 **更多** 按钮
3. 选择 **设为默认队列**

### 默认队列标识

- 默认队列在列表中显示 ⭐ 星标图标
- 每个用户只能有一个默认队列
- 设置新的默认队列会自动取消原默认队列的标识

---

## 🗑️ 删除队列

### 操作步骤

1. 在队列列表中找到要删除的队列
2. 将鼠标悬停在队列上，点击右侧的 **更多** 按钮
3. 选择 **删除**
4. 在确认对话框中点击 **删除** 确认

### 注意事项

- ⚠️ 删除队列会同时删除队列中的所有消息
- ⚠️ 删除操作不可撤销
- 建议在删除前先处理或归档重要消息

---

## 📊 队列统计

### 未读消息数

- 每个队列右侧显示未读消息数量
- 侧边栏顶部显示所有队列的未读消息总数
- 导航栏的待办入口也会显示未读角标

### 消息状态统计

在消息列表中可以通过筛选器查看不同状态的消息：

| 状态 | 说明 |
|------|------|
| **未读** | 新收到的消息 |
| **已读** | 已查看但未处理的消息 |
| **处理中** | 正在处理的消息 |
| **已处理** | 已完成处理的消息 |
| **已归档** | 已归档的消息 |

---

## 💡 最佳实践

### 队列分类建议

根据不同用途创建多个队列：

| 队列类型 | 用途 | 可见性建议 |
|----------|------|------------|
| **工作任务** | 接收工作相关的任务消息 | 公开 |
| **个人备忘** | 保存个人备忘和待办事项 | 私有 |
| **学习资料** | 收集学习相关的内容 | 私有 |
| **团队协作** | 接收团队成员的协作消息 | 公开 |

### 队列命名规范

- 使用简洁明了的名称
- 避免使用特殊字符
- 建议使用英文或拼音作为队列名称
- 显示名称可以使用中文

### 定期整理

- 定期处理未读消息
- 将已完成的消息归档
- 删除不再需要的消息
- 清理不再使用的队列
