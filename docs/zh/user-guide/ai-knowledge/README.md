# 📚 AI 知识库

AI 知识库模块提供文档存储、组织和检索功能，实现 RAG（检索增强生成），让 AI 智能体能够基于您的特定文档回答问题。

## 🎯 核心功能

### 文档管理

支持多种文档来源和格式：

- **文件上传**：支持 PDF、Markdown、Word、TXT 等格式
- **文本粘贴**：直接创建文本文档
- **网页抓取**：从 URL 导入内容
- **外部表格**：导入钉钉/飞书表格

### 智能分块

自动将文档分割成适合检索的最佳片段：

- **Markdown 文件**：按标题和句子分割
- **文本文件**：按句子边界分割
- **PDF/Word**：递归字符分割

### RAG 检索

支持多种检索模式：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **向量** | 语义相似度搜索 | 自然语言查询 |
| **关键词** | 传统 BM25 文本搜索 | 精确词条匹配 |
| **混合** | 向量 + 关键词组合 | 综合最佳效果 |

### 引用机制

AI 响应包含链接到源文档的编号引用：

```
根据您的产品文档 [1]，该功能的工作方式是...
此外，故障排除指南 [2] 建议...

来源：
[1] product-overview.pdf（第 3 节）
[2] troubleshooting-guide.md（分块 15）
```

## 🚀 快速开始

1. **创建知识库**：前往知识库页面，点击"创建知识库"
2. **上传文档**：添加您的文档（PDF、Markdown 等）
3. **配置检索**：选择检索器和嵌入模型
4. **开始对话**：在聊天中选择知识库，开始提问
