# 消息处理

本文档介绍如何查看和处理待办队列中的消息。

---

## 📋 消息列表

### 页面布局

待办页面分为两个区域：

- **左侧边栏**：队列列表，显示所有待办队列和未读消息数
- **右侧主区域**：消息列表，显示当前选中队列的消息

### 消息卡片

每条消息以卡片形式展示，包含以下信息：

| 元素 | 说明 |
|------|------|
| **状态指示器** | 显示消息状态（未读显示蓝点） |
| **发送者** | 转发消息的用户名 |
| **时间** | 消息接收时间 |
| **优先级** | 高优先级显示红色图标，低优先级显示蓝色图标 |
| **备注** | 发送者添加的备注说明 |
| **内容预览** | 消息内容的前两行预览 |
| **消息数量** | 转发的消息条数 |

---

## 🔍 筛选和排序

### 状态筛选

点击状态筛选下拉框，可以按消息状态筛选：

| 状态 | 说明 |
|------|------|
| **全部** | 显示所有消息 |
| **未读** | 仅显示未读消息 |
| **已读** | 仅显示已读消息 |
| **处理中** | 仅显示正在处理的消息 |
| **已处理** | 仅显示已完成处理的消息 |
| **已归档** | 仅显示已归档的消息 |

### 排序方式

点击排序下拉框，可以选择排序方式：

| 排序 | 说明 |
|------|------|
| **最新优先** | 按时间倒序，最新的消息在前 |
| **最早优先** | 按时间正序，最早的消息在前 |

---

## 📖 查看消息详情

### 打开详情

点击消息卡片，打开消息详情对话框。

### 详情内容

消息详情对话框显示：

- **发送者信息**：用户名和邮箱
- **接收时间**：消息的接收时间
- **状态标签**：当前消息状态
- **优先级标签**：消息优先级（非普通优先级时显示）
- **备注说明**：发送者添加的备注
- **消息内容**：完整的对话内容，包括用户消息和 AI 回复

### 消息内容展示

- 用户消息和 AI 消息有不同的背景色
- 显示每条消息的发送者和时间
- 支持滚动查看长对话
- 显示消息附件信息（如有）

### 自动标记已读

- 打开消息详情时，未读消息会自动标记为已读
- 未读消息数会实时更新

---

## ⚡ 处理消息

### 处理按钮

在消息详情对话框或消息卡片的更多菜单中，点击 **处理** 按钮开始处理消息。

### 处理流程

1. 点击 **处理** 按钮
2. 系统跳转到对话页面
3. 消息内容作为上下文传递
4. 可以选择智能体继续处理

### 适用场景

- 需要基于消息内容进行进一步操作
- 需要使用 AI 智能体处理消息
- 需要与消息内容相关的后续对话

---

## 📝 消息操作

### 单条消息操作

点击消息卡片右侧的更多按钮（三个点图标），可以进行以下操作：

| 操作 | 说明 |
|------|------|
| **处理** | 开始处理消息，跳转到对话页面 |
| **标记为已读** | 将未读消息标记为已读 |
| **标记为未读** | 将已读消息标记为未读 |
| **归档** | 将消息归档 |
| **设置优先级** | 修改消息优先级 |
| **删除** | 删除消息 |

### 批量操作

#### 进入批量选择模式

1. 点击消息列表顶部的 **选择** 按钮
2. 进入批量选择模式
3. 勾选要操作的消息

#### 批量操作选项

| 操作 | 说明 |
|------|------|
| **标记为已读** | 批量标记选中的消息为已读 |
| **标记为未读** | 批量标记选中的消息为未读 |
| **归档** | 批量归档选中的消息 |
| **处理** | 批量处理选中的消息 |
| **删除** | 批量删除选中的消息 |

#### 全选和取消

- 点击顶部的复选框可以全选/取消全选当前页的消息
- 显示当前选中的消息数量

#### 退出批量模式

点击右上角的 **X** 按钮退出批量选择模式。

---

## 🏷️ 消息状态

### 状态说明

| 状态 | 图标 | 说明 |
|------|------|------|
| **未读** | 🔵 蓝色实心圆点 | 新收到的消息，尚未查看 |
| **已读** | ⚪ 空心圆点 | 已查看但未处理的消息 |
| **处理中** | 🔄 旋转图标 | 正在处理的消息 |
| **已处理** | ✅ 绿色勾选 | 已完成处理的消息 |
| **已归档** | 📦 归档图标 | 已归档的消息 |

### 状态流转

```
未读 → 已读 → 处理中 → 已处理
         ↓
       已归档
```

---

## ⚠️ 删除消息

### 单条删除

1. 点击消息卡片右侧的更多按钮
2. 选择 **删除**
3. 在确认对话框中点击 **删除** 确认

### 批量删除

1. 进入批量选择模式
2. 勾选要删除的消息
3. 点击 **删除** 按钮
4. 在确认对话框中点击 **删除** 确认

### 注意事项

- ⚠️ 删除操作不可撤销
- ⚠️ 删除后消息将永久移除
- 建议重要消息先归档而非删除

---

## 💡 最佳实践

### 高效处理

- 定期查看未读消息
- 优先处理高优先级消息
- 使用筛选功能快速定位消息
- 处理完成后及时更新状态

### 消息整理

- 定期归档已处理的消息
- 删除不再需要的消息
- 保持队列整洁有序

### 团队协作

- 及时处理收到的转发消息
- 处理完成后可以回复发送者
- 建立团队的消息处理规范
