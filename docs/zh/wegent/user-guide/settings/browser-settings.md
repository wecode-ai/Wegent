# 浏览器设置

Wework 桌面版在 **设置 → 集成 → 浏览器** 中管理内置浏览器。Codex 浏览器工具控制的是当前任务右侧面板中同一个可见页面，不会另外打开 Chrome 窗口。

## 链接打开位置

普通网页链接与本地开发链接可以分别选择打开位置：

- **系统默认浏览器**：交给操作系统设置的默认浏览器。
- **Wework**：在当前任务绑定的内置浏览器面板中打开。

如果当前没有可用的任务浏览器面板，Wework 会回退到系统默认浏览器，避免链接无响应。

本地链接包括 `localhost`、`*.localhost`、`127.0.0.0/8`、`0.0.0.0` 和 IPv6 回环地址。默认情况下，普通网页使用系统浏览器，本地开发链接使用 Wework。

## 浏览数据与隐私

内置浏览器使用独立于 Wework 主界面的数据存储。清除浏览数据会移除内置浏览器的 Cookie、缓存和网站存储，但不会清除 Wework 应用登录状态。

Wework 不读取或导入 Chrome 中保存的密码、联系信息和个人资料。需要使用现有 Chrome 登录态时，请把对应链接的打开位置设为系统默认浏览器。这个边界避免应用复制或另行保存浏览器凭据。

## 下载

下载默认保存到系统下载文件夹。可以选择自定义目录，也可以开启 **下载前询问保存位置**，让每次下载都显示系统保存对话框。

下载开始后，内置浏览器工具栏会自动展开下载栏，并显示实时进度。下载过程中可以暂停或继续；暂停后可以删除任务和未完成文件。下载完成后，可以从下载栏查看文件名并在 Finder 中显示文件。
