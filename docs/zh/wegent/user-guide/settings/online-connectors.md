# 在线连接器

Wework 可以让本地编程任务使用 OpenAI 提供的在线连接器，例如 GitHub 或 Google Drive。在线连接器仅支持通过 ChatGPT 登录，不支持 API Key；具体服务仍需单独登录和授权。

## 首次导入 Codex 配置

首次启动时，如果 Wework 检测到本机已有 Codex 配置，会询问是否导入其中的插件、技能和插件市场信息。这个步骤只处理本地配置，不会启用在线连接器。

选择 **仅创建新配置** 会保留原 Codex 目录并创建独立的 Wework 配置；选择 **导入已有配置** 会复制可迁移的本地内容。两种选择都会保持在线连接器关闭。

## 启用在线连接器

打开 **设置 → 集成 → 插件**，在 **在线连接器** 中开启 **加载在线连接器**。开启后，Wework 会从 OpenAI 获取当前 ChatGPT 账号可用的连接器。关闭开关不影响本地插件、技能或普通编程功能。

在线连接器是否可用还取决于 ChatGPT 工作区权限和管理员策略。即使连接器已经显示，GitHub、Google Drive 等第三方服务也不会自动授权；首次使用时仍需按服务要求登录并确认访问权限。
