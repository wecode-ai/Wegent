# 快速开始

## 安装与启动

从团队提供的下载地址或 [Wegent Releases](https://github.com/wecode-ai/Wegent/releases) 获取适合当前系统的 Wework 安装包。完成安装后启动应用。

首次启动时，Wework 会准备本机执行环境。如果电脑上已有 Codex 配置，可以按引导导入；也可以稍后在 **设置 → 模型** 中完成 Codex 登录或配置其他模型。

## 创建第一个任务

1. 在左侧选择 **新对话**。
2. 在输入区选择可用模型。
3. 输入一个目标明确的任务，例如“分析这个项目的目录结构，并说明本地启动方法”。
4. 发送消息并等待执行结果。

没有选择项目时，对话使用独立工作区。需要让 AI 处理已有代码时，先创建项目并选择代码目录。

## 打开本地项目

1. 点击项目区域旁的添加按钮。
2. 选择 **使用现有目录**。
3. 在系统目录选择器中选中项目根目录。
4. 打开项目，在输入框中描述任务。

项目添加到 Wework 后，磁盘上的原目录和文件保持在原位置。

## 推荐的第一个编码任务

```text
请先阅读项目说明和贡献规范，然后：
1. 说明实现这个需求会涉及哪些模块；
2. 完成修改；
3. 运行相关测试；
4. 总结改动和仍需关注的风险。
```

首次处理重要仓库时，建议先让 AI 阅读项目规范，再开始修改。
