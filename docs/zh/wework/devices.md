# 设备与云端工作

## 本机工作

Wework 启动后会管理本机执行环境。本地项目、会话、Codex 和本地模型可以直接在本机使用。关闭窗口并进入系统托盘后，正在执行的本机任务会继续运行；完全退出应用会停止 Wework 管理的本机执行进程。

## 连接 Wegent

1. 点击侧边栏中的 **连接云端**。
2. 输入团队提供的 Wegent Backend 地址。
3. 点击 **授权并连接**。
4. 在授权页面完成登录并确认授权 Wework。

连接后，工作台会显示服务端模型、云设备和远程设备。断开连接不会删除本地项目和本地会话。

## 云设备

创建项目或任务时，可以选择在线的云设备。代码目录、终端和任务命令都位于所选设备。设备处于忙碌、离线或需要升级状态时，界面会给出相应处理入口。

## 远程 Docker 设备

在 **设置 → 连接 → 添加设备** 中选择远程 Docker 设备，复制生成的启动命令并在目标机器执行。设备上线后可为其选择项目目录并运行任务。

启动命令包含设备凭证，应通过安全渠道保存和使用。

## 在其他设备继续任务

使用任务菜单中的 **复制到其他设备**，选择目标设备和项目目录。正在运行的任务会先停止当前回复，再创建目标设备上的副本。
