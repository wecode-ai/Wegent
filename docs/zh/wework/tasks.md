# 与 AI 协作

## 描述任务

一条有效的任务说明通常包含目标、范围、约束和验收方式。例如：

```text
为用户设置页增加语言选择：
- 支持中文和英文；
- 复用现有设置组件；
- 选择后立即生效并持久化；
- 补充相关单元测试。
```

复杂任务可以先要求 AI 调研并给出方案，确认后再实施。

## 添加上下文

你可以在输入区添加文件、图片、代码位置和应用快照。应用快照会截取当前前台窗口，并在获得系统辅助功能权限时附带窗口文本。

发送本地图片后，Wework 会在消息中保留图片预览。关闭并重新打开 Wework 或切换到其他会话后，返回原会话仍可查看该图片；如果原始本地文件已被删除，预览将无法恢复。

输入 `/` 可以选择可用 Skill。插件安装后，其 Skills、命令或工具会出现在对应任务能力中。

## 选择模型和设备

模型决定任务使用的 AI 能力，设备决定代码和命令在哪里执行。本地模型只能用于本机设备；云端模型和设备需要先连接 Wegent。

打开输入区的模型选择器后，如果当前只有一个模型分组，Wework 会直接展示该分组中的全部模型；如果存在多个分组，请先选择分组，再从展开的二级菜单中选择模型。

## 任务运行中的控制

任务运行时有三种发送方式：

| 操作               | 快捷键                         | 适用场景                   |
| ------------------ | ------------------------------ | -------------------------- |
| 当前回复结束后发送 | `Enter`                        | 补充下一步要求             |
| 引导当前回复       | `Command/Ctrl + Enter`         | 调整正在进行的工作方向     |
| 打断并立即发送     | `Command/Ctrl + Shift + Enter` | 当前方向错误，需要马上停止 |

打断会停止当前回复，但已经完成的文件修改和命令不会自动回滚。

## 管理会话

任务可以继续对话、重命名、复制到其他设备或归档。项目会话显示在所属项目下；没有项目的会话显示在独立对话列表中。已归档会话可以在设置中恢复或删除。

当你在消息底部切换到其他会话时，Wework 会保留“跟随最新消息”的状态。即使原会话在后台继续输出，切回后仍会显示最新内容；如果你在切换前主动向上滚动，Wework 则会恢复原来的阅读位置。
