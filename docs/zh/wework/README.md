# Wework 文档

Wework 是面向 AI 协作的桌面工作台。你可以直接打开本地项目，让 Codex 在项目目录中分析代码、修改文件和运行命令；也可以连接 Wegent，使用云端模型和执行设备。

## 从这里开始

1. 按照[快速开始](./getting-started.md)完成首次配置并创建第一个任务。
2. 阅读[项目与工作区](./projects.md)，选择本地目录、Git 项目或独立对话。
3. 通过[与 AI 协作](./tasks.md)了解附件、Skills 和运行中的任务控制。
4. 在[代码工作台](./workbench.md)中查看文件、终端和代码变更。
